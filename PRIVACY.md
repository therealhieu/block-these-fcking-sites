# Privacy Policy — blockthesefckingsites

**Last updated:** 2026-03-18

## Data Collection

blockthesefckingsites does **not** collect, transmit, or share any personal data.

All user data — including blocked site lists, profile settings, schedule rules, and temporary unlock timestamps — is stored locally on the user's device using `chrome.storage.local`. No data ever leaves the browser.

## Permissions

| Permission | Why it's needed |
|---|---|
| `storage` | Store blocked sites, profiles, and schedule settings locally |
| `webNavigation` | Detect when the user navigates to a blocked site |
| `tabs` | Redirect open tabs that match blocked domains |
| `<all_urls>` | Check any URL against the user's blocklist (no page content is read) |

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking.
